Course Overview

In this course, you will develop a solid foundation in HTML and CSS. You will learn how to create web pages using HTML5 and advanced CSS libraries such as Bootstrap. The course will cover using existing jQuery plugins or writing your own to discover new ways of adding interactivity, animation, and better usability. You will create, design, and implement web layouts focused on design principles, and utility.


Course Learning Outcomes

After completing this course, you will be able to:

  1. Describe HTML and CSS concepts
  2. Utilize HTML5 and advanced CSS
  3. Demonstrate the plan for jQuery plugins and implementation of interactivity, animation, and better usability
  4. Apply design principles and utility in the creation, design, and implementation of web layouts using Bootstrap
