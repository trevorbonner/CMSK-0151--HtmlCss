In Class Excercise


We are going to create this page in class. All the instructions required are also going to be on this page with a few blanks. This will be an empty link.


Step By Step:

  1. Create new file called index
  2. add the doctype tag and open and close of html tag
  3. add the head and body tags
  4. add and empty style tag to the head
  5. add the h1 tag with text
  6. add the p tag with text, embed the anchor tag to create a link
  7. add the h3 tag for steps
  8. add the ordered list tag and populate it with the required items
  9. add the h3 tag for items that can be done in any order
  10. add the unordered list tag and populate it with the required items

Can be done in any order:

  • add br tags where necesary
  • to the stlye tag add a body section and give it a background-color, i went with cornflowerblue but you can pick your own.
  • set the margin in the body to keep it away from the edges. I choose 20px
  • for your paragraph and list item elements give them a color of your choosing, a font-size of your chosing as well
  • Just for the paragraph element set text-align to be center
  • for the anchor tags choose a color. Also for the anchor tag set the text-decoration to be _blank
  • for the anchor tag set the target to be none
  • for your header tags pick a different font-family and give them a color as well
  • give your headers a text-decoration, I gave mine underline
